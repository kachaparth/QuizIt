# QuizIt 

QuizIt is a secure, customizable quiz and examination platform** designed for colleges and institutions.  
It supports online, offline (LAN-based) quizzes with real-time control, advanced analytics, and ML-assisted cheat detection**.

---

## Features

- Fully customizable quiz creation  
  - Per-question time limits  
  - Multiple question types (MCQ, MSQ, Numeric, Coding, etc.)
- Multiple quiz modes  
  - Server-controlled live quizzes  
  - Randomized question order  
  - Offline / LAN-based quizzes
- Anti-cheating mechanisms  
  - Tab switch & focus detection  
  - Fullscreen enforcement  
  - Camera & microphone-based monitoring
- ML-assisted analytics  
  - Behavioral anomaly detection  
  - Cheating risk scoring  
  - Answer similarity analysis
- Detailed post-quiz analytics  
  - Participant performance  
  - Question-wise insights  

---

## Tech Stack

- **Backend:** Spring Boot, JPA (Hibernate)  
- **Database:** PostgreSQL (Supabase)  
- **Frontend:** React / Next.js  
- **Realtime:** WebSockets  
- **ML:** OpenCV / MediaPipe, Behavioral Analysis Models  

---

## Use Cases

- College examinations  
- Internal assessments  
- Competitive quizzes  
- Offline classroom tests  

---

## Project Goal

To provide a reliable, offline-capable, and secure quiz platform with intelligent monitoring, suitable for real-world academic environments.
